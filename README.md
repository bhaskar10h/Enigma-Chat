<h2>🛠️ Running Steps:</h2>

<p>1. open your command-terminal</p>

```
cd "Project destination"
```

<p>2. Now run the server.py script to start</p>

```
python server.py
```

<p>3. After running the server.py on another terminal run the client.py script</p>

```
python client.py
```
